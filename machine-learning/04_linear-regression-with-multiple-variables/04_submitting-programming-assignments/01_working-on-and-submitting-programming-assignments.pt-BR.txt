nos seus exercícios de Aprendizado de Máquina. Aqui está minha janela do Octave e,
primeiro vamos para o meu desktop. Eu salvei os arquivos para o meu primeiro exercício,
alguns deles no meu desktop: nesse diretório, 'ml-class-ex1'. E nós fornecemos um número de arquivos
e pedimos pra você editar alguns deles. Assim, para o primeiro arquivo, voce vai encontrar detalhes
no arquivo pdf para esse exercício de programação. Mas, um dos arquivos que pedimos para você editar
é esse, chamado "warmUpExercise.m", onde o exercício é apenas para ver se você está
familiarizado com o sistema de envio. E tudo o que você precisa fazer é
retornar a matriz identidade "5x5". Portanto, a solução desse exercício, eu vou
mostrar, é so escrever "A = eye(5)". Assim, isso modifica a função para
gerar a matriz identidade "5x5". E essa função "warmUpExercise()"
agora retorna a matriz identidade "5x5" E eu vou salvar. Então, eu fiz a primeira parte da tarefa.
Voltando para a janela do Octave, vamos para o diretório,
'C:\Users\ang\Desktop\ml-class-ex1'. E, se eu quiser ter certeza que isso foi implementado,
eu digito 'warmUpExercise()', e isso retorna a matriz identidade "5x5"
