Mari bicara tentang regresi logistik. Di video ini, saya ingin memperlihatkan hipotesis, yakni fungsi yang akan kita gunakan untuk mewakili hipotesis untuk kasus klasifikasi. Sebelumnya, kita ingin pengklasifikasi menghasilkan nilai antara 0 dan 1. Karena itu, kita ingin hipotesis yang sesuai dengan syarat tersebut, yakni prediksinya
di antara 0 dan 1. Ketika kita menggunakan regresi linier, ini model hipotesisnya, dimana h(x) = (theta)T x.
Untuk regresi logistik, saya akan memodifikasi sedikit, dan membuat hipotesisnya g((theta)T x), dimana fungsi g sebagai berikut: g(z) jika z angka riil sama dengan 1/(1+e^-z). Ini disebut fungsi sigmoid atau fungsi logistik. Dari istilah fungsi logistik itulah muncul istilah regresi logistik. Istilah fungsi sigmoid dan fungsi logistik pada dasarnya sinonim dan punya arti yang sama. Jadi, keduanya dapat dipertukarkan dan bisa digunakan untuk menunjuk fungsi g ini. Jika kita ambil 2 persamaan ini, dan menggabungkannya, ini alternatif lain menuliskan model hipotesis saya. h(x) = 1/(1 + e^-((theta)T x)) h(x) = 1/(1 + e^-((theta)T x)) h(x) = 1/(1 + e^-((theta)T x)) Yang saya lakukan adalah saya mengganti variabel z, z angka riil, dengan (theta)T x. Jadi, saya mengganti z dengan (theta)T x. Saya akan tunjukkan tampilan
fungsi sigmoid. Akan saya plot di gambar ini. Fungsi sigmoid, g(z), juga disebut fungsi logistik, tampak
seperti ini. Mulai dari dekat nol, lalu naik hingga mencapai 0.5 dari titik asal dan kemudian
mendatar seperti itu. Begitulah tampilan fungsi sigmoid. Perhatikan fungsi sigmoid g(z) asimtotnya di 1. Bila sumbu horisontal z mencapai minus tak terhingga g(z) mendekati 0, dan bila z mencapai tak terhingga, g(z) mendekati 1. Oleh karena g(z) bernilai antara 0 dan 1, maka h(x) pasti bernilai antara 0 dan 1. Dengan hipotesis seperti ini, yang perlu dilakukan, yaitu mencocokkan parameter theta dengan
data kita. Jadi, dengan set latihan yang ada, kita pilih nilai parameter theta, dan hipotesis ini akan membuat prediksi bagi kita. Kita akan bicarakan nanti algoritma belajar untuk mencocokkan
parameter theta. Untuk saat ini, mari bicara tentang interpretasi model ini. Begini cara saya akan menafsirkan hasil dari hipotesis h(x). Saat hipotesis menghasilkan beberapa angka,
saya akan memperlakukan angka itu sebagai dugaan probabilitas bahwa y = 1 pada input baru misalnya x.
Ini maksud saya. Ini sebuah contoh. Katakanlah kita menggunakan contoh
klasifikasi tumor. Jadi, kita punya ciri-ciri vektor x, dimana x0 = 1 dan x1 adalah ukuran tumor. Anggap saya punya pasien, dan punya beberapa ukuran tumor.
Saya masukkan ciri-ciri vektor x itu ke hipotesis saya dan anggap hipotesis menghasilkan angka 0.7. Saya akan tafsirkan hipotesis saya sebagai berikut. Hipotesis ini sedang mengatakan pada saya bahwa pasien dengan ciri-ciri x,
kemungkinan y = 1 adalah 0.7. Dengan kata lain,
saya akan katakan pada pasien saya bahwa kemungkinan tumornya ganas adalah 70% atau 0.7. Untuk menuliskan ini sedikit lebih formal atau secara matematika,
saya akan menafsirkan hasil hipotesis saya sebagai h(x)= p( y=1 | x; theta) h(x)= p( y=1 | x; theta) Jadi, bagi Anda yang akrab dengan
probabilitas, persamaan ini dapat dimengerti, bila Anda kurang akrab dengan probabilitas, begini cara saya membaca persamaan ini. Ini probabilitas dimana y = 1, dengan x yaitu pasien saya punya ciri-ciri x. Pasien saya punya ukuran tumor yang diwakili oleh ciri-ciri x, dan parameternya theta. Jadi, pada dasarnya saya akan bergantung pada hipotesis untuk memberikan perkiraan kemungkinan y = 1. Sekarang, karena ini tugas klasifikasi, y harus bernilai 1 atau 0, kan? Itu adalah dua kemungkinan nilai y, entah dalam set latihan atau bagi pasien baru yang berjalan menuju kantor saya atau kantor dokter
dimasa depan. Jadi, dengan h(x) kita bisa juga menghitung probabilitas y = 0. Konkritnya, karena y harus bernilai 0 atau 1, probabilitas y = 0 ditambah y = 1 harus sama dengan 1. Persamaan pertama ini tampak sedikit rumit, tapi itu dasarnya menyatakan bahwa
probabilitas y = 0 untuk pasien tertentu dengan ciri-ciri x, dengan parameter theta, ditambah probabilitas y = 1 untuk pasien yang sama dengan
ciri-ciri x dan parameter theta, harus sama dengan 1.
Jika persamaan ini tampak sedikit rumit, bayangkan saja persamaan itu
tanpa x dan theta. Ini menyatakan probabilitas P(y = 0) + P(y = 1) = 1. Kita tahu ini benar karena y harus bernilai
0 atau 1. Jadi, kesempatan y = 0 ditambah kesempatan y = 1 harus sama dengan 1. Jadi, jika Anda ambil bagian ini dan pindahkan ke sebelah kanan, maka hasilnya persamaan ini yakni P(y = 0) =  1 - P(y = 1).
