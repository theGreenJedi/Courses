Бұл видео арқылы сіз Питон бағдарламасын қалай жүтеп алуға, және бұл бағдарламаның не туралы екендігі жөнінде ақпатар береді.
Бағдарламалар сіздің компьютерлеріңізде және интернетте таралған.
Сіз веб браузерін қолданасыз, компьютер ойындарын ойнайсыз, ұялы телефоныңызда мәтіндік хабарлама жазасыз, осы уақыттың барлығында сіз 
бағдарлама қолданасыз. Тіпті сіздің OS10, сияқты оперативтік жүйеңізде Linux немесе  Windows бағдарламалар болып табылады. бағдарлама дегеніміз нұсқаулықтардың топтамасы.
Бағдарламалар орындалады немесе қосылады. Сіздің операциялық жүйеңіз сияқты бағдарлама ұялы телефон сияқты немесе ноутбук немесе планшет секілді құрылғыларда қолданылады.
Дегенмен бағдарламалар басқа бағдарламар арқылы да орындалуы мүмкін.
Бұл сіз ойын ойнаған кезде болатын веб браузердегі амал.
Сіздің веб браузеріңіз Java script деп аталатын бағдарлама тілінде қалай жазылатындығын біледі. википедиядағы осы веб беттердегі бағдарламалау тілдерін барлығына қолжетімді және олар мұнда өте көп. Сіз сондай бағдарламалау тілдерінің бірқатарын естіген шығарсыз. Біз олардың көбісін естідік бірақ олардың бірі туралы анық ақпаратты естіген жоқпыз және осы жерде тізілген бағадрламалау тілдерінің барлығын біреу біледі дегенге күмәнданамын. Сонымен Питон өзгелер сияқты бағдарламалау тілі болып табылады. Питон бағдарламалау тілін үйрену үшін сіз сіздің браузеріңіз Javascript қалай түсінсе Питонды тура солай түсінетін бағдарлама жүктеп алуыңыз қажет. Осы курс барысында жұмыс атқару үшін сіз осы бағдарламаны python.org ресурсынан жүктеп алғаныңыз жөн.
Питонның екі негізгі нұсқасы бар. Питон екі және Питон үш.
Біз Питон үш нұсқасын қолданбақшымыз. Одан әрі жүріп Quick Links амалын таңдаңыз да Питонға арналған бағдарламаның үшінші нұсқасын жүктеп алыңыз.
Егер сіз Linux қолдансаңыз, сіз Питон үшті жүктеуге көмектесетін жүктегішті енгізуіңізге болады.
Егер сіз Windows қолдансаңыз, осы сілтеме дұрыс.
Ал егер сіз OS ten қолдансаңыз соңғы нұсқаулары болса, онда бұл жерден жүктеп алуға болады. Егер сіз Mac қолдансаңыз сізге TickleTK амалын жүктеп алған жөн, ол питон бағдарламасын жазуға көмектесетін бағдарлама. Осы жердегі түймені басып көрейік.
Егер сіз Mac пайдаланушысы болсаңыз осы сілтемені басып, Active TCLE.5, .12 амалын немесе оның ең жаңа нұсқасын жүктеп алыңыз.
Сіз жергілікті вебсайтқа кіріп мына үлкен көкті басып жүктеп алу үшін осы амалды орындағаныңыз жөн. Біздің Питон бағдарламамызды жазу үшін біз Idle деп аталатын Монти Питонның Эрик Идл атына берілген атауына басып Питонды жүктеп аламыз. Суретке екі рет басқаннан кейін сіз
