Este vídeo mostrará a você como baixar
o Python e também comentará um pouco sobre é um programa.
Programas estão por toda parte em seu computador e na internet.
Cada vez que você usa um navegador de web, joga um videogame ou escreve uma mensagem de texto em
seu celular, você está usando um programa. Mesmo seu sistema operacional, seja ele o OS10,
Linux ou Windows, é um programa. Um programa é apenas um conjunto de instruções.
Dizemos que programas são executados. Programas, tal como seu sistema operacional,
são executados sobre um hardware, como um celular, ou seu laptop ou tablet.
Mas programas podem também ser executados por outros programas.
Isto é o que acontece quando você brinca com um jogo em um navegador web.
Seu navegador web sabe como executar programas escritos em uma linguagem de programação chamada
Javascript. Esta página web da Wikipedia lista todas as
linguagens de programação que alguém tenha já preocupado em entrar aqui, e existem
centenas delas. Você provavelmente já deve ter ouvido falar
em algumas das linguagens de programação. Nós já ouvimos falar em muitas delas, mas
nem de longe, em todas elas e duvidamos que que alguém saiba todas as linguagens
de programação listadas aqui. Python é uma linguagem de programação,
uma de muitas. A fim de aprender a programar em Python,
você precisa baixar um programa que compreenda essa linguagem, da mesma forma
que seu navegador entende Javascript. Para trabalhar neste curso, você
precisará baixar este programa do python.org
