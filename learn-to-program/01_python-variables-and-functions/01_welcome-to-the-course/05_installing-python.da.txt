Denne video viser, hvordan du kan hente
Python og fortæller lidt om, hvad et program er. Programmer findes 
overalt på din computer. og på internettet. Hver gang du bruger 
en browser, spiller et computerspil eller sender en SMS på 
din mobiltelefon, anvender du et program Selv operativsystemer, som OS10, Linux 
eller Windows er programmer Et program er en sekvens af instruktioner. 
Programmer køres eller eksekveres. Programmer som dit operativsystem kører 
på hardware, som en mobiltelefon eller din bærbar, eller tablet. 
Men programmer kan også køres af andre programmer. Det er det, der sker,
når du spiller et spil i en browser. Din browser er 
i stand til at køre programmer skrevet i et programmeringssprog kaldet 
Java Script. Denne side på wikipedia viser alle de 
programmeringssprog, som brugere har brugt tid på at anføre, 
og der er 100 vis af dem. Du har sikker hørt om
en del af disse programmeringssprog. Vi kender til mange af dem, men 
langt fra alle, og vi tvivler på at der er nogen, der kender alle 
programmeringssprogene anført her. Så, Python er et programmeringssprog, 
et blandt mange. For at kunne lære Python, er det nødvendigt
at hente et program, der Forstår sproget, på samme vis, 
som din browser forstår Java Script. Til dette kursus, skal du hente 
dette program fra python.org.
Der findes to primære udgaver af Python. Python to og Python tre.
Vi vil anvende Python tre. Klik på Quick Links, og
på denne side, kan du hente Python tre, den seneste version.
Hvis du bruger Linux, kan du bruge pakke manageren til at installere Python tre.
Hvis du bruger Windows, skal du bruge dette link. Hvis du bruger OS10,
af en af de senere versioner, er dette det 
rigtige link at bruge Hvis du bruger Mac, skal du evt.
installere TickleTK for at kunne køre det program, vi anvender til at skrive
Python programmer. Lad os scrolle ned i bunden af siden,
hvis du bruger Mac klik på dette link, scroll ned og hent Active 
TCLE.5, .12, eller hvad den seneste udgave nu er. 
Du skal en tur forbi active state hjemmesiden og klik så på den blå
download knap for at hente programmet. Til at skrive Python programmer anvender
