Acest video vă arată cum să descărcați
 Python și vorbim puțin și despre ce este un program. Programele se găsesc
 peste tot în computerul vostru și pe internet. De fiecare dată când
 utilizați un browser web, jucați un joc pe calculator sau scrieți un mesaj text
 pe telefonul mobil, utilizați un program. Chiar și sistemul vostru de operare, ca
 OSX, Linux sau Windows, este un program. Un program este doar un set de instrucțiuni.
 Programele sunt rulate sau executate. Un program, cum ar fi sistemul de operare,
 rulează pe hardware, ca telefonul mobil, sau laptop-ul sau tableta. Dar programele
 pot fi rulate și de alte programe. Asta se întâmplă
 când jucați un joc într-un browser web. Browser-ul web
 știe cum să ruleze programe scrise într-un limbaj de programare
 numit JavaScript. Această pagină web de pe Wikipedia listează
 toate limbajele de programare pe care cineva s-a obosit vreodată să le introducă
 aici, și sunt sute de limbaje. Probabil ați auzit de unele dintre
 limbajele de programare. Noi am auzit de multe dintre ele, dar cu siguranță
 nici pe departe de toate și ne îndoim că cineva cunoaște toate limbajele
 de programare listate aici. Deci, Python este un limbaj de
 programare, unul dintre multe altele. Pentru a rula programe Python, trebuie să
 descărcați un program care înțelege acest limbaj, așa cum
 browser-ul web înțelege JavaScript. Pentru a urma acest curs, va trebui să
 descărcați acest program de pe python.org. Există două versiunii
 principale de Python, Python 2 și Python 3.
 Noi vom utiliza Python 3. Deci, dați clic pe Quick Links și aceasta
 este pagina de descărcare pentru Python 3, versiunea curentă. Dacă folosiți Linux,
 puteți utiliza gestionarul de pachete pentru a instala Python 3.
 Dacă folosiți Windows, acest link este cel potrivit. Dacă folosiți OSX, una dintre versiunile recente, cel puțin, acesta
 este link-ul potrivit. Dacă utilizați un Mac, va trebui să instalați
 ceva numit Tcl/Tk pentru a rula programul pe care îl folosim pentru
 a scrie programe Python. Deci, să coborâm aici, la sfârșitul paginii.
 Dacă folosiți un Mac, dați clic pe acest link, defilați în jos și descărcați
 Active Tcl 8.5.12 sau care este versiunea cea mai recentă. Va trebui să fiți
 nevoiți să vă duceți la pagina web de status activ și să apăsați pe butonul
 Download, mare și albastru, pentru asta. Pentru a scrie programele Python vom
 folosi un program, IDLE, numit după Eric Idle din Monty Python.
 Acesta vine odată cu Python. Doar dați dublu clic pe pictogramă și sperăm
 că veți vedea o fereastră care arată așa.
