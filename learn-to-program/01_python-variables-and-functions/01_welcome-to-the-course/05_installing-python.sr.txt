U ovoj epizodi pokazaćemo kako da skinete Python i
ukratko započeti priču o tome šta je to program.
Programi su svuda. Kako na vašem kompjuteru tako i svuda po internet-u.
Svaki put kada pokrenete web pretraživač, startujete neku kompujutersku igru ili započnete poruku 
na vašem mobilnom - vi koristite neki program. Vaš operativni sistem, na primer OS 10,
Linux ili Windows, je takođe program. Program je samo niz instrukcija.
Za programe obično kažemo da se izvršavaju. Programi poput operativniog sistema
se izvršavaju na nekom hardware-u, kao što su mobilni telefon, laptop ili tablet.
Ali programe takođe mogu da izvršavaju i neki drugi programi.
To je situacija kada igrate neku igru u web pretraživaču.
Vaš web pretraživač zna kako da izvršava programe napisane u programskom jeziku zvanom
Java Script. Ova stranica na wikipedia-i sadrži spisak svih
programskim jezika koje je neko smatrao za shodno da trebaju da budu upisani.
I ima ih na stotine. Do sada ste verovatno već čuli
za neke programske jezike. Nama su poznati mnogi od njih.
Ali zasigurno ne i većina ostalih. Sumnjam da postoji iko ko poznaje
sve programske jezike na ovom spisku. Tako je i Python, u mnoštvu ostalih, jedan od programskih jezika. Kako bi ste savladali Python programiranje
neophodno je da skinete program koji razume taj jezik isto kao što
web pretraživač razume Javascript. Kako bi ste mogli da napredujete kroz ovaj kurs
neophodno je da skine ovakav program sa python.org
Tamo ćete naći dve glavne verzije Python-a. Python verziju 2 i Python verziju 3.
Mi ćemo da koristimo Python 3. Slobodno idite na "Quick Links" sekciju.
Tu je "download" stranica za skidanje trenutno aktuelne podverzije Python-a 3.
Ako koristite Linux, pokrenite omiljeni menadžer paketa i instalirajte paket za Python 3.
Ako koristite Windows ovaj link je sasvim korektan.
Ako koristite OS 10, tj neku od skorašnjih verzija OS 10, onda je za vas
ovo pravi link. Ako koristite Mac računar moraćete takođe
da instalirate TickleTK, kako bi koretno radio program u kome ćemo pisati
naše nove python programe. Ako siđemo naniže, pri dnu se nalazi link za korisnike Mac računara. Ovde se nešto niže nalazi download link za ActiveTcl 8.5.12 ili koja je već aktuelna podverzija. Otići ćete na Active state web sajt i izabrati
ovo veliko pravo dugme za download. Da bi pisali naše nove python programe koristićemo program sa imenom "Idle". Ime je dobio po glumcu Eric Idle iz serije Monty Python.
Naći ćete ga kao deo postojeće instalacije Python-a. Pokrenite ga duplim klikom na ikonicu
i trebalo bi da dobijete prozor koji ovako izgleda.
