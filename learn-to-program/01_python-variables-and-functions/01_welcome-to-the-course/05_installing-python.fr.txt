Cette vidéo vous montre comment télécharger
Python et parle un peu de ce qu'est un programme.
Les programmes sont partout sur votre ordinateur et sur Internet.
Chaque fois que vous utilisez un navigateur, jouez à un jeu vidéo ou écrivez un message depuis
votre téléphone portable, vous en utilisez un. Même votre système d'exploitation, tel qu'OS10,
Linux ou Windows est un programme. Un programme est simplement un jeu d'instructions.
On dit qu'un programme tourne ou est exécuté. Un programme comme votre Système d'Exploitation
tourne sur du matériel, tel qu'un téléphone ou votre ordinateur portable, ou une tablette.
Mais un programme peut aussi être exécuté par d'autre programmes.
C'est ce qui se produit quand vous jouez à un jeu vidéo dans un navigateur.
Votre navigateur sait comment exécuter des programmes écris dans un language de programation
appelé JavaScript Cette page de Wikipédia liste tous les
languages que les utilisateurs du site ont décidé d'enregistrer, et il y en
a des centaintes. Vous avez probablement entendu parler
de certains d'entre eux. Nous avons entendu parler de beaucoup d'entre eux
mais pas de tous et nous doutons sincèrement que quelqu'un connaisse tous les languages
listés ici. Python est donc un language de programmation,
un parmis tant d'autres. Pour apprendre à programmer en Python,
vous devez télécharger un programme qui comprends ce language tout comme
votre navigateur comprends JavaScrit Pour cette classe, vous devrez
installer ce programme depuis le site python.org.
