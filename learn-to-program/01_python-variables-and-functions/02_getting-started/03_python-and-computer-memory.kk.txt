Мысалы, электронды таблицалау программасы сандар тобының орташасын есептегенде, ең біріншіден программа барлық сандардың қосындысын табады, сосын сандардың санын есептеп, сәйкесінше бөлуді орындайды. Барлық мәндер компьютер жадында сақталады. Осы курстың мақсаты үшін, компьютер жадын сақтау аймағының ең ұзын тізімі ретінде қарастырмақпыз, оның әрбірі арнайы жад деп аталатын бірегей санға ие. Арнайы жадты біз, әдетте, өзгеден ажырату үшін Х префиксі көмегімен жазамыз, яғни x201 бұл 201 үшін арныйы жад. Осы арнайы жадты ұзын көше бойында орналасқан үйлердің саны ретінде қарастырсаңыз болады. Айтқанымыздай, мәндер компьютер жадында сақталған. Осыған орай, 8.5, x34 арнайы жадында сақталуы мүмкін, x34,ал 44 саны x35 арнайы жадында сақталуы мүмкін. Барлық мәндерді сақтау үшін программаға жол қажет. Оны айнымалы деп аталатын мән ретінде жүзеге асырады. Айнымалы жадтағы атаулы мекен. Питон айнымалыларды жадтың мәндерді сақтайтын бөлігінен өзге аймақта сақтайды. x34 арнайы жадында сақталатын әртүрлі аяқ киім өлшемдері бар яғни бұл аяқ киім өлшемі 8.5 мәніне ие дегенді білдіреді. Дәл осылай, Полдың жасы болатын айнымалыға ие боуымыз мүмкін, оның мәні 44. Біз бұны x35 арныйы жадына Полдың жасы болатын айнымалыны сақтау арқылы жүзеге асырамыз. Pythonның ыңғайлығы, сізге қайсы арнайы жады таңдалып алынғандығы жөнінде ойлану қажет емес. Python сіз үшін соны орындайды. Шынында, сақтау үшін арнайы жадын таңдай алмайсыз. Дегенмен, бұл сіз программалауды жүзеге асырып жатқанда осы моделді есте сақтауда жақсы көмектеседі. Курс барысында қолданатын терминология ұсынылған. Мән арнайы жадқа ие. Айнымалы ішінде айнайы жады бар. Айнымалы мәнге қатысты немесе мәнді білдіреді. Осы терминологияны қолданудың бірнеше мысалы көрсетілген. 
8.5 мәні x34 арнайы жадына ие. Аяқ киім өлшемі айнымалысы ішінде x34 арнайы жады бар. Аяқ киім өлшемі мәні 8.5ке тең.
