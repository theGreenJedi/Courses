ordenador. Para el propósito de este curso, pensaremos
en la memoria del ordenador como una muy larga lista de localidad de almacenamiento, cada cual
que tiene un número único llamada la
dirección de la memoria Usualmente escribiremos en la dirección de la memoria
con un prefijo x de de tal forma que se vean diferente de otros números, x201 es la
dirección de memoria 201. Puedes pensar de ellas como dirección de memoria como los
números de casa en una larga calle. Como hemos dicho, los valores son almacenados en 
la memoria del ordenador. Así el número 8.5 puede estar en la
dirección de memoria x34 x34, y el número 44 puede estar en la
dirección de memoria x35 Los programas necesitan una forma de mantener un registro de todos
estos valores Ellos hacen esto usando algo llamado
variables Una variable es una ubicación con nombre en la memoria Python mantiene un registro de las variables en un
área separada de la memoria de los valores
Podemos tener una variable tamaño de zapatos que almacena dirección de memoria x34 lo que significa que
el tamaño de zapato se refiere al valor 8.5.
De la misma forma, podríamos tener la variable paul_age cuyo valor fue 44
Esto lo logramos almacenando la dirección de memoria x35 con
la variable paul_age. Lo bueno de Python es que tu
no tienes que preocuparte sobre que dirección de memoria particular
son escogidas Python va a mantener un registro de todas estas por
ti De hecho, nunca podrás saber qué dirección
de memoria se está usando. Sin embargo, es realmente una ayuda mantener
ese modelo en mente cuando estás programando. Estos son algunos términos que usaremos
durante el curso. Un valor tiene una dirección de memoria Una variable contiene una dirección de memoria Una variable se refiere o apunta a un valor
Aquí hay unos ejemplos de como se usa esta terminología.
El valor 8.5 tiene la dirección de memoria x34 La variable shoe_size contiene la dirección de memoria
x34. El valor de shoe_size es 8.5.
shoe_size se refiere al valor 8.5 shoe_size apunta al valor 8.5.
