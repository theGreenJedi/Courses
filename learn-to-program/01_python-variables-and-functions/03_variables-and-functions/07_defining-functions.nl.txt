Hallo. Naast het gebruik van Python's ingebouwde functies, kunnen we ook zelf functies definiëren en daar gaat deze les over. Laten we beginnen met een wiskundig voorbeeld. Functie F van X neemt X en vermenigvuldigt dat in het kwadraat. Zo ziet de functie eruit in Python. Def geeft aan Python aan dat we een functie definiëren. F is de naam van de functie. X wordt de parameter van de functie genoemd. Dit een variabele wiens waarde meegeleverd wordt als de functie wordt opgeroepen. 
Dit dubbelpunt maakt Python duidelijk dat wat hierna komt hetgeen is wat zal gebeuren als de functie uitgevoerd wordt
