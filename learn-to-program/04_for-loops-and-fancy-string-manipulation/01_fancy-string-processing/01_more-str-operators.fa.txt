دیده اید که چگونه دو string را در هم ادغام میکنیم و چگونه میتوانید که string را در عدد صحیح ضرب کنید تا تکرار شود. در این ویدئو،در مورد سایر انواع عملگرهای string تحقیق میکنیم. برای متغیرsolution عبارت cat را معین کردیم. اکنون،از علامت برابر برای مقایسه و دیدن این که آیا solution به cat مربوط میشود یا خیر استفاده میکنیم. علامت نابرابر هم هست که این گونه است با یک علامت تعجب و نشانه برابر. حل مساله را بهstring جدیدی متوجه میسازیم. آنگاه همان عملیات را بررسی میکنیم. دو string را میتوانیم برای ترتیب دیکشنری در اینجا،پایتون با شروع از اول حرف به حرف مقایسه میکند. آن عملگر کمتر از بود. و نیز عملگر بزرگتر است از نیز دارد. به همین گونه عملگرهای کمتر است از یا برابر است با و بزرگتر یا مساوی با داریم میتوانیم حروف بزرگ را باهم مقایسه کنیم. در واقع،تبدیل به حرف بزرگ اهمیت دارد،و حروف بزرگ از حروف کوچک کمترند. هر حرفی را که بتوانید تایپ کنید قابل مقایسه است. همیشه میتوان دو مقداراز انواع گوناگون را برای برابری و نابرابری مقایسه کرد،ولی معمولا نمیتوان دو آیتم مختلف برای ترتیب مقایسه نمود. اینجا عملگر جدیدی برای string هست. میتوان دید که آیا string درون string دیگر هست یا نه. با استفاده از این،میتوان کنترل کرد که آیا c حرف صدا دار هست یا نه. ترتیب حروف اهمیت دارد. Zoo در ooze نیست. میتوان پرسید که آیا string تهی داخل string دیگر هست،و پاسخش بله است، و نیز string تهی داخل خودش است. آخرین مطلبی را که در این مبحث مورد بحث قرار میدهیم تابع است،نه عملگر.
