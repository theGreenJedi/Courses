Ernest Hemingway escreveu: "por toda minha vida 
olhei as palavras como se as tivesse vendo pela primveira vez". 
Palavras estão por toda parte na programação. Das buscas do Google à Biotecnologia, 
onde DNA é, às vezes, representado como um emaranhado de As, Ts, Cs e Gs, à coisas
 simples, com um diálogo. Agora mesmo, exploraremos a representação de 
strings do Python, e como podemos manipular estes valores. 
Uma string literal é uma sequência de caracteres. 
No Python, este tipo é chamado str. Strings iniciam e terminam com aspas simples ou 
duplas, como 'hello' ou "how are you". Podemos incluir pontuações, 
como acabamos de ver e hifens.
Strings são valores e podem ser usadas em uma declaração de atribuição. 
Se está um dia ensolarado, podemos decidir dizer para alguém "What a beautiful day!". 
E se estiver chovendo muito, poderíamos querer dizer, "Wow, you're dripping wet".
Mas, já podemos ver que temos um um problema nesta frase, porque vai da cor verde à 
preta no segundo apóstrofo e, de fato, temos um erro de sintaxe aqui. 
O problema, é claro, é que estamos tentando usar uma aspa simples dentro de uma 
string com aspas simples, e o Python, claro, realmente não sabe o 
que estamos fazendo. Então, por causa disso, o Python permite que você
use de aspas duplas. Então, veja: a saudação ensolarada, sunny_greeting está assim,
e a saudação chuvosa, storm_greeting, corretamente, toda molhada! Temos outra escolha. 
Em vez de usar aspas duplas nesta string, podemos usar o que chamamos de 
"Escape Character" uma sequência de escape. Se usarmos uma barra invertida, 
você notará que a string ficou toda verde. E quando eu pressiono Enter, o que nós 
veremos é que esta string aparecerá, aparecerá, mas não veremos a barra invertida. Além disso, a aspa simples, 
no início e no fim da string, se tornará aspas duplas na exibição. 
Podemos concatenar strings usando o operador mais " + ". 
Note que temos que incluir os espaços. Com isso, podemos escrever uma versão de MatLibs,
onde pedimos uma palavra a um amigo. Digamos que eles escolham minhoca. Então, teremos que iniciar um quebra-cabeças. E gostaríamos de finalizar isso com um sinal de exclamação 
e a palavra escolhida por eles, 'birthmark'. Agora, vamos colocá-las juntas. Podemos construir uma string 
repetida usando o operador de multiplicação " * ". E, como regra básica de matemática, 
multiplicação precede a adição. Podemos sobrescrever usando parênteses. No entanto, não podemos adicionar 
uma String a um Int ou a um Float. (digitando) E converter, implicitamente, um objeto Float em Str,
soa um pouco perigoso. Note que é um tipo de erro. 
Você verá, nas próximas aulas, como combinar strings e números. Também não podemos multiplicar strings, 
mesmo que a string contenha números. Não podemos multiplicar uma sequência de Str. Outros operadores não funcionarão. Ocorrerão: "Erros de tipo"; "Tipos de operandos não  suportados para ' - '  Str";  e "Str não pode usar duas strings juntas com subtração". Não se pode dividir tipos de operandos não suportados por divisão 
Para elevar a uma potência. Note que cada um deles é um "Erro de Tipo" de variável.
