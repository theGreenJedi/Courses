Como hemos visto, Python tiene una
función incluída llamada help -ayuda- que les mostrará información sobre una función.
La interface Idle también provee información cuando comienzan a escribir una llamada a
esa función. Por ejemplo, cuando llaman a la función help
en la funció valor absoluto, obtienen una linda descripción de lo que hace.
También, cuando comienzan a escribir una llamada a esa función, aparece un recuadrito amarillo
en Idle mostrando los tipos de parámetros y el tipo que devuelve.
Ustedes pueden proveer el mismo tipo de información en sus propias funciones,
escribiendo documentación en un formato particular llamado una cadena doc -doc string-.
Hemos escrito la documentación para la función del área de un triángulo que escribimos la semana
pasada. Noten que estamos usando una cadena entre comillas
triples porque nuestra descripción se extiende en múltiples líneas.
Ahora correremos este módulo y vamos a tener la función area  disponible en Phyton,
y cuando pedimos ayuda en area, ahora vemos lo que escribimos en la cadena doc. Es más, esta primera línea de la
cadena de ducumentación es lo que aparecerá cuando comencemos a escribir una llamada a area.
De ahora en adelande, siempre preoveeremos cadenas doc para nuestras funcions, tanto para
describir lo que hacen, como para enganchar en el sistema de ayuda de Python.
