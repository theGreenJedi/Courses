Las listas son secuencias mutables.
También hay secuencias inmutables. Éstas se llaman Tuples -tuplas-.
Aquí creamos una tupla con A3 y -0.2 en ella, y la asignamos a la variable tup.
Las tuplas y listas usan la misma notación para los índices.
Aquí tenemos el índice cero de la tupla, índice uno, índice dos, y así sucesivamente, también podemos usar
índices negativos, como índice menos uno de tup.  Cortar una tupla produce otra tupla. Aquí, coramos la tupla tup hasta dos, pero no
incluímos el índice dos y también podemos cortar tup desde el índice uno hasta, pero no incluyendo, el índice tres. No podemos cambiar las tuplas, Las tuplas son inmutables. Asignar a una tupla produce un error de
tipo -type error, tuple object does not support item assignment- el objeto tupla no soporta asignación. 
Las listas tienen los métodos añadir, contar, extender, -append, count, extend- indizar, insertar, soltar, remover, invertir y ordenar
-index, insert, pop, remove, reverse, sort- Pero las tuplas sólo tienen contar e indizar.
Todos los otros métodos de las listas modifican la lista.
Como en las listas, podemos iterar sobre los elementos de una tupla. La función len -longitud- funciona en las tuplas. Podemos también iterar sobre los índices de una
tupla Para crear una tupla, pongan una lista de valores
separados por comas entre paréntesis. Y qué pasa con las tuplas de un elemento? Si simplemente
ponemos un valor entre paréntesis, es decir ponemos entre paréntesis esta expresión
matemática, y nos devuelve el valor. Para crear una tupla de un elemento, hay que poner una coma después del valor adentro del paréntesis.
 Las tuplas vacías no requieren coma.
